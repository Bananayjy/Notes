## 一、docker占用端口查看

查看docker的守护进程

```
[root@hs-redis ~]# systemctl status docker
● docker.service - Docker Application Container Engine

```

`Main PID` 是指 Docker 服务进程的主进程 ID。在你提供的输出中，`Main PID: 7701 (dockerd)` 表示主进程的进程号是 `7701`，并且这个进程是 `dockerd`，也就是 Docker 守护进程。这个进程负责启动和管理 Docker 容器的运行。

Docker 守护进程通常占用一个端口（如 TCP 端口 `2375`）用于远程访问，但它本身并不一定仅占用一个端口。
